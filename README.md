# Whirl
A variety of activity indicators for apps of any kind.

### Indicator Types:
- Normal
- Labrinth

### How to install:
```swift
pod 'Whirl'
```

### How to use:
```swift
import Whirl
```

```swift
//--Initialize Indicator
let indicator = Indicate(indicatorType: .normal)

//--Start Indicator
indicator.startIndicator()

//--Stop Indicator
indicator.stopIndicator()
```
